II. Identificación de peligros y riesgos ambientales
En este elemento es importante identificar los riesgos que conlleva cada actividad dentro de una estación de servicio, así mismo como el impacto ambiental que este podría tener. Una vez identificados, es importante evaluar estos riegos para determinar un nivel de peligro y así poder tomar acciones preventivas o correctivas.
